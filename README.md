# 链尚ant-design-vue后台管理系统的开发框架模版

> 基于vue,vuex,vue-router,ant-design-vue,axios的开发框架

## 使用文档

使用文档请参考[向日葵系统](http://git.lsfash.cn/f2e/vue-omniscient)(公司内网)

## 用法

``` bash
$ npm install -g vue-cli
$ vue init lianshang-f2e/antd my-project
$ cd my-project
$ npm install
$ npm run dev
```

### 构建自己的项目

``` bash
vue init lianshang-f2e/antd my-project
```
